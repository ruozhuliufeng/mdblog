## 项目需求
### 博客分类
- 公开博客：允许游客浏览，允许登陆用户评论与点赞，参与博客排名
- 保护博客：允许游客浏览，允许登陆用户评论与点赞，允许登陆用户进行编辑，同时保存编辑记录，参与博客排名
- 私人博客：不允许游客浏览，不开放编辑，不参与博客排名
